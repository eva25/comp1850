Two HTML tags that have not been discussed inside the classroom.

The Sup and Sub Tags in HTML

The Subscript Tag:

This tag is used to define subscript text. Subscript can be defined by using the following tags <Sub> ... </Sub>

Example:

This text is subscripted!

Code:

<p>This text is <sub>subscripted!</sub></p>



The Subsperscript Tag:

This tag is Used to define superscript text. Tags that are used to define superscript are.. <Sup> ... </Sup>

Example:

This text is superscripted!

Code:

<p>This text is <sup>superscripted!</sup></p>
