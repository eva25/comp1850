Hover Effect with CSS

So basically on this tutorial all we have to do is manipulate an image behaviour by hovering it. This tutorial is very simple and very amusing I learned this from this site: Fivera.net, they also have some other tutorials on how to create your website to become even more interactive, though I only want to focus on this hover effect. So lets get started!

Prepare the followings:

For HTML
  1. Image that can be use.
  2. And of course this tag:<img src="http://link to the image">
For CSS it can either be external or internal
  1. img {here is the code of the initial state of the image} and;
  2. img:hover { here is the code of the hover state of the image}

Shrink Image on Hover

HTML codes:
<div style="text-align:center">
<img src="../images/s1.png" alt="fivera" width="200" height="200">

CSS codes:
img {
height: 200px;
width: 200px;

-webkit-transition: all 1s ease;
-moz-transition: all 1s ease;
-o-transition: all 1s ease;
-ms-transition: all 1s ease;
transition: all 1s ease;
}

img:hover {
width: 220px;
height: 220px;
}
